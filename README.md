# Project-Donations

Sistema de gerenciamento de doações

Iniciamos a modelagem pelo Doador pois ele é o ator principal.

Doador:
- nome
- email
- cpf
- endereço
- telefone
- histórico de doações

Doação:
- valor
- doador
- projeto

Projeto:
- nome
- email
- cpf ou cnpj
- endereço
- telefone
- equipe
- custo mês
- orçamento

Orçamento:
- contas a pagar
  + aluguel
  + água
  + luz
  + internet
- equipe
- infra

Como queremos rastrear a Doação até seu pagamento no Orçamento
